# kotlin-web-api-spring-maven-mongo-select

## Description
Creates an employee if collection is empty
or has more than 5 employees.

## Tech stack
- kotlin
- maven
  - mongo connector
  - gson
- mongo

## Docker stack
- mongodb:latest
- openjdk11

## Notes


## To run
`sudo ./install.sh -u`
Available at http://localhost/employee

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
